# Spor Salonu Yönetim Sistemi (FitnessApp)

## Proje Hakkında
Bu proje, Web Programlama dersi kapsamında geliştirilen bir **Spor Salonu Yönetim ve Randevu Sistemidir**.
ASP.NET Core MVC kullanılarak geliştirilmiştir.

## Özellikler
- Antrenör ve Hizmet Yönetimi
- Üye Randevu Sistemi (Çakışma Kontrolü ile)
- Admin Paneli
- REST API Raporlama
- Yapay Zeka Destekli Egzersiz Önerileri
